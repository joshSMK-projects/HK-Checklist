Project Name
    Hollow Knight Checklist

Project Description
    This project allows users to store checkpoints in the game Hollow Knight using
    LocalStorage. The percentage matches the games percentage of 112%.

How to run the project
    1. Download the project
    2. Run the index.html file

How to use the project
    1. Run the game Hollow Knight
    2. As you play the game mark up your progression in the app
    3. When finished or starting a new game press the reset button
